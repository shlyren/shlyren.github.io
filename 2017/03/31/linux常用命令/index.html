    1. clean命令删除所有的软件安装包

      • 在网络连接正常的情况下，我们执行软件安装命令，软件安装结束后，以 .deb 为后缀的软件包就不再需要了。所以要对他定期清理

      • 使用上面命令即可上错所有的软件包

    2. autoclean 删除不再可用的安装包

      • 与clean一样, 只不过它有选择地删除软件包：那些不再可用的安装包(比如，你再也不能够从软件仓库中下载到该软件包的当前版本、或更新版本)。

    3. remove 删除特定软件

    4. purge清除软件残余

      • purge 命令是 remove 命令的另一个版本，它可以在删除软件本身的同时，删除软件包的残余设置。
