使用gem更新报错的问题

Dec 2, 2016 | error

gem install cocoapods

使用gem install cocoapods更新报You don't have write permissions for the /Library/Ruby/Gems/2.0.0 directory错.
解决方法: 使用sudo gem install cocoapods 更新

sudo gem install cocoapods

使用sudo gem install cocoapods更新报Operation not permitted - /usr/bin/pod错

解决方法: 使用sudo gem install -n /usr/local/bin cocoapods
